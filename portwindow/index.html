jquery port window

I know, I know. I've seen images presented into something like this before or somewhere. Then again, I wanted to stretch myself a bit away from work and do some simple scripts with jquery - it's a bit crude and straight-forward, but usable. Resize the browser window it the image should follow to a degree. Degrades into a simple box in IE6; but still borked by min-width/min-height usage. Check-out the script and let me know any suggestions/improvements.

jQuery finds images with the class "port" and uses the alt and title attributes to fill-out the left bar details: 

<img src="[image]" class="port" alt="[used as heading] title="[used as commentary]" />

-Kev
